# Create-folder-from-TXT

Batch folder creation based on a TXT file

# Instructions

1. Download Python

2. Ir a la carpeta contenedora del Proyecto con:
   
   ```Python
   cd "Directorio Objetivo"
   ```

3. Execute script 
   
   1. Windows:
      
      ```Python
      python "Creador de carpetas.py"
      ```
   
   2. Linux/Mac
      
      ```Python
      python3 "Creador de carpetas.py"
      ```
